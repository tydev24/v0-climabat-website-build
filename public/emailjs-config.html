Configuration EmailJS (Optionnel)

Pour améliorer l'envoi d'emails, vous pouvez configurer EmailJS :

  1. Allez sur https://www.emailjs.com/
  2. Créez un compte gratuit
  3. Configurez un service email (Gmail, Outlook, etc.)
  4. Créez des templates pour "contact" et "appointment"
  5. Remplacez les clés dans lib/form-service.ts
  6. Ajoutez ce script dans app/layout.tsx :

<script
  type="text/javascript"
  src="https://cdn.jsdelivr.net/npm/@emailjs/browser@4/dist/email.min.js"
></script>
<script type="text/javascript">
  (function() {
    emailjs.init("YOUR_PUBLIC_KEY");
  })();
</script>

Sans EmailJS : Les formulaires ouvriront automatiquement votre client email avec le message pré-rempli.
